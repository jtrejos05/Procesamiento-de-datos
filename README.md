# Taller Python
Este repositorio contiene el desarrollo del Taller de Ejercicios en Python, el cual tiene como objetivo aplicar y reforzar los conceptos fundamentales del lenguaje mediante la resolución práctica de actividades organizadas en cuadernos (notebooks).

Cada cuaderno aborda un tema específico y presenta ejercicios desarrollados con su respectiva explicación, ejecución y conclusiones. El repositorio está estructurado de manera ordenada para facilitar la navegación, revisión y comprensión del contenido.

<b>Objetivos del Taller</b>

*Aplicar los conceptos básicos y fundamentales de Python.

*Desarrollar habilidades en lógica de programación.

*Implementar estructuras de datos y control de flujo.

*Documentar adecuadamente los procesos y algoritmos utilizados.

*Presentar soluciones claras, organizadas y funcionales.
